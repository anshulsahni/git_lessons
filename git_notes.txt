- commit - smallest unit of work
- branches - collection of commits
   - default branch -> name of the branch is "main"
   - but you can create as many branches as possible

git init - initialise the repository
git add <file_name> - add the file to stage
git commit -m <commit message> - will commit the file
git log - list all the commits of the current branch in decreasing order
git status - gets the current status of all the files
git diff - checks the difference in content of changed files
